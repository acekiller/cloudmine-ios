typedef void(^ CMStoreDeleteCallback)(BOOL success)

Callback block signature for all operations on CMStore that delete objects or binary files. This block should return void and take a single boolean as an argument. It will be YES if the objects/files were deleted successfully, and NO otherwise. If files were not successfully deleted, check CMStore::lastError for details.

See also:
- deleteObject:callback: (CMStore)

typedef void(^ CMStoreFileFetchCallback)(CMFile *file)

Callback block signature for all operations on CMStore that fetch binary files from the CloudMine servers. This block should return void and take a single CMFile as an argument. This will contain the data as well as a bit of metadata about the file that was downloaded.

typedef void(^ CMStoreFileUploadCallback)(CMFileUploadResult uploadResult)

Callback block signature for all operations on CMStore that upload binary files to the CloudMine servers. This block should return void and take a single CMFileUploadResult as an argument. This will the result of the upload (namely, whether the upload created a new file or updated an old one).
